# Vision-Driable-Area-Lane-Detection-ONNX
Convert the SOTA driable area and lane detection models into ONNX format.


### **支持的模型 Supported Models**
1. [TwinLiteNetPlus](https://github.com/chequanghuy/TwinLiteNetPlus)


### **性能 Performance**  
| OS           | Device       | Backend           | Model        | Time Cost in Seconds <br> (720 x 1280 pixels) |
|:------------:|:------------:|:-----------------:|:------------:|:------------------------------------------------:|
| Ubuntu-24.04 | Laptop       | CPU <br> i7-1165G7 | TwinLiteNetPlus-Large <br> f32 | 0.25                          |
| Ubuntu-24.04 | Laptop       | GPU <br> MX150 | TwinLiteNetPlus-Large <br> f32 | 0.21                              |
